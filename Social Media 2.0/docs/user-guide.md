# SocialSphere User Guide

## Authentication
- Use the auth form to sign in.
- Use "Need an account? Sign up" to create a new account.

## Feed
- Write a post and click `Publish`.
- Click `♥` to like/unlike a post.
- Click `💬` to open comments on a post.
- Add a comment using the comment input and `Send`.
- Use `Edit` and `Delete` on your own posts.
- Feed updates are real-time (new posts, likes, comments).

## AI Features
- `Polish Text`: rewrites your current draft.
- `Suggest Idea`: fills the input with a topic idea.
- `Summarize`: summarizes current feed topics.

## Profile and Network
- Open profile from the navbar or from user cards.
- Edit your own profile and click `Save Profile`.
- Use `Back to me` after viewing another user's profile.
- Follower and following counts are shown on profile cards.
- Use `Send Request` to request following another user.
- Use `Accept` or `Reject` for incoming follow requests.
- Use `Unfollow` or `Cancel Request` to remove follow relations.

## Search
- Use the `Search by name or handle` input in the sidebar.
- Search filters the currently loaded user suggestions.
